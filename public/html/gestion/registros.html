Lista de registros

Sección con la lista de registros pendientes de validar en el sistema.

La lista puede filtrarse por el tipo de registro (objeto, tumba o inhumación). Para ello, seleccionar el tipo que se desee, y después pulsar Buscar registros.

También se puede buscar un registro en concreto mediante su identificador. Para ello introducir el identificador y se filtrarán los resultados.

Si se quieren ver todos los elementos de nuevo, pulsar Ver todo.

Pulsar Ver para ver la información de un elemento.
Pulsar Gestionar para editar un elemento.
Pulsar Validar para verificar un elemento.
