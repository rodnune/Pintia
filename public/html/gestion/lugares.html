Gestión de lugares.

Sección para crear/gestionar las lugares.

Seleccionar definir nueva zona ,rellenar los campos del formulario y pulsar Agregar para guardar la nueva zona arqueológica.

Seleccionar una zona existente y pulsar Guardar cambios para editar la zona.

Seleccionar una zona existente y pulsar Eliminar para borrar la zona del sistema.

Los campos con son obligatorios.
