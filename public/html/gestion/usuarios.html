Lista de usuarios

Sección para gestionar usuarios.

La lista puede filtrarse por el rol del usuario(Regular,Novel,Experto y Administrador). Para ello, seleccionar el rol que desee y después pulsar Buscar usuarios.

También se puede buscar un usuario en concreto mediante su username. Para ello introducir el identificador de usuario y se filtrarán los resultados.

Si quiere añadir un nuevo usuario, pulsar Añadir usuario

Para actualizar los datos de un usuario pulsar Gestionar y rellenar el formulario.

Si quiere borrar la cuenta de un usuario, pulsar Eliminar
