Gestión de medidas.

Sección para crear/gestionar las medidas.

Seleccionar definir una nueva medida ,rellenar los campos del formulario y pulsar Crear medida para guardar una nueva medida.

Seleccionar una medida existente y pulsar Guardar cambios para editar la medida.

Seleccionar una medida existente y pulsar Eliminar medida para borrar la medida del sistema.

Los campos con son obligatorios.
