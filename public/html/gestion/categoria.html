Categoría.

Sección para gestionar una categoría

Pulsar Modificar Nombre para actualizar la denominación de la categoría.

Pulsar Eliminar para borrar la categoría del sistema.

Pulsar Asociar para asociar una medida a la categoría.

Pulsar Eliminar asociación para borrar la asociación con la medida seleccionada.

Si se desea gestionar una subcategoría, pulsar en la parte inferior Gestionar para editar la subcategoría.
