Lista de UE

Sección con la lista de todas las UE del sistema.

Pulsar Ver para acceder a los datos de la UE seleccionada.

La lista de UE puede filtrarse por diferentes parámetros como los componentes geológicos, artificiales u orgánicos.
Para ello, seleccionar aquellos parámetros de búsqueda que se deseen, y después pulsar Buscar UE para buscar UE que cumplan esos parámetros.

También se puede buscar una unidad estratigráfica en concreto mediante su identificador. Para ello introducir el identificador y se filtrarán las ue que contengan dicho identificador.
Si se quieren ver de nuevo todas las UE disponibles, pulsar Ver todo

Para editar una UE debes de ser usuario experto o superior.
