Relaciones Estratigráficas

Sección para realizar/eliminar asociaciones entre UE.

Selecciona la UE actual y la UE que se quiere relacionar. Después selecciona el tipo de relación que hay entre ellas y pulsa Asociar para añadir la relación entre las UE.

Para eliminar una relación entre UE, seleccionarla y pulsar Eliminar asociación

Ten en cuenta que cuando asocias dos UEs se registran las relaciones en ambos sentidos.
