Lista de Muestras

Sección con la lista de todas las muestras del sistema.

La lista puede filtrarse por tipos de muestra Para ello, seleccionar el tipo de muestra deseado, y después pulsar Buscar muestras para buscar muestras que cumplan esos parámetros.

También se puede buscar muestra en concreto mediante su número de registro. Para ello introducir el número y se filtrarán la(s) muestra(s) que contengan ese registro.

Si se quieren ver de nuevo todas las muestras disponibles, pulsar Ver todo

Para editar,crear o borrar una muestra debes de ser usuario de nivel experto o mayor
