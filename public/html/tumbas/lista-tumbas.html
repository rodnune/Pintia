Lista de tumbas

Sección con la lista de todas las tumbas del sistema.

Pulsar Ver para acceder a los datos de la tumba seleccionada.

La lista de tumbas puede filtrarse por diferentes parámetros como el año, tipo de tumba o lugar. Para ello, seleccionar aquellos parámetros de búsqueda que se deseen, y después pulsar Buscar tumbas para buscar tumbas que cumplan esos parámetros.
También se puede buscar una tumba en concreto mediante su identificador. Para ello introducir el identificador y se filtrarán las tumbas que contengan dicho identificador.
Si se quieren ver de nuevo todas las tumbas disponibles, pulsar Ver todo
