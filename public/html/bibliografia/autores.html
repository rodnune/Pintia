Lista de autores.

Sección con la lista de autores existentes.

Para crear un nuevo autor, pulsar Nuevo autor.

Para ver información del autor, pulsar Ver .

Para editar un autor, pulsar Gestionar Autor.

Para borrar un autor del sistema, pulsar Eliminar.

Se puede buscar un autor concreto mediante su nombre. Para ello introducir el nombre y se filtrarán los resultados

Si se quieren ver todos los autores, pulsar Ver todo
