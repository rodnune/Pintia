Lista de artículos.

Sección con la lista de articulos existentes.

Para crear un nuevo articulo, pulsar Nuevo.

Para ver información del articulo, pulsar Ver .

Para editar un artículo, pulsar Gestionar.

Para borrar un artículo del sistema, pulsar Eliminar.

La lista de artículos puede filtrarse por diferentes parámetros como el autor o la palabra clave.

Para ello, seleccionar aquellos parámetros de búsqueda que se deseen, y después pulsar Buscar Artículos para buscar artículos que cumplan esos parámetros.
También se puede buscar artículos en concreto mediante su título. Para ello introducir el título y se filtrarán los artículos que lo contengan.

Si se quieren ver todos los artículos, pulsar Ver todo
