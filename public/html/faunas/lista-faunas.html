Lista de analíticas de faunas.

En esta sección se puede ver la lista de analíticas de faunas existentes.

Se puede buscar una analítica en concreto mediante su identificador. Para ello introducir el identificador y se filtrarán la(s) analítica(s) que coincidan con dicho indentificador

Si se quieren ver de nuevo todas las analíticas disponibles, pulsar Ver todo.

Para editar, borrar y crear una analítica de faunas debes ser un usuario de nivel experto o mayor
