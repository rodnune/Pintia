Lista de inhumaciones

Sección con la lista de todas las inhumaciones del sistema.

Pulsar Ver para acceder a los datos de la inhumación seleccionada.

La lista de inhumaciones puede filtrarse por diferentes parámetros como la UE de cadáver, fosa, estructura o relleno, o la tumba (una tumba puede tener cero o más inhumaciones asociadas).

Para ello, seleccionar aquellos parámetros de búsqueda que se deseen, y después pulsar Buscar inhumaciones para buscar inhumaciones que cumplan esos parámetros.
También se puede buscar una inhumaciones en concreto mediante su identificador. Para ello introducir el identificador y se filtrarán las inhumaciones que contengan dicho identificador.
Si se quieren ver de nuevo todas las inhumaciones disponibles, pulsar Ver todo

Para editar una inhumación debes ser propietario de esta o ser un usuario de nivel superior al del creador de la inhumación.
