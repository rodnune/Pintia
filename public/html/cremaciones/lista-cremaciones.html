Lista de cremaciones

Sección con la lista de cremaciones del sistema.

Pulsar Ver para acceder a los datos de la cremación seleccionada.

La lista de cremaciones puede filtrarse por diferentes parámetros como la UE, sexo o la tumba (una tumba puede tener cero o más cremaciones asociadas)
Para ello, seleccionar aquellos parámetros de búsqueda que se deseen, y después pulsar Buscar cremaciones para buscar cremaciones que cumplan esos parámetros.

También se puede buscar una cremación en concreto mediante su código propio. Para ello introducir el código y se filtrarán las cremaciones que contengan dicho código.
Si se quieren ver de nuevo todas las cremaciones disponibles, pulsar Ver todo.
