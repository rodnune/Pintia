Lista de objetos

Sección con la lista de todos los objetos del sistema.

Pulsar Ver para acceder a la descripción del objeto seleccionado.

La lista de objetos puede filtrarse por diferentes parámetros como el año, material o lugar. Para ello, seleccionar aquellos parámetros de búsqueda que se deseen, y después pulsar Buscar objetos para buscar objetos que cumplan esos parámetros.

También se puede buscar un objeto en concreto mediante su referencia. Para ello introducir la referencia y después pulsar Buscar objeto

Si se quieren ver de nuevo todos los objetos disponibles, pulsar Ver todo
